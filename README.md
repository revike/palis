Установка и запуск
------------------

* Переходим в директорию проекта

```cd palis```

* Создаем файл .env такой же, как .env.sample (меняем настройки при необходимости)

```touch .env```

###### Делаем файл docker_commands.sh исполняемым

```chmod +x docker_commands.sh```

##### Запуск

```docker-compose up -d --build```
